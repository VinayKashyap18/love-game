# Love Game 💖

A premium, real-time connection game for couples.

## Setup Instructions

### Backend
1. Open a terminal in the `backend` folder.
2. Run `npm install` (if not already done).
3. Run `go-live.bat`. 
   - *Note: To enable internet play, make sure you have an internet connection. The script uses Localtunnel to generate a public link.*

### Frontend
1. Open a terminal in the `frontend` folder.
2. Run `npm install` (if not already done).
3. Run `npm run dev`.
4. Open the link displayed (usually `http://localhost:5173`).

## Playing
- **Host**: Click "Create New Room". Share the 6-digit code with your partner.
- **Partner**: Enter the 6-digit code and click "Join Room".
- Choose your roles and start the game!
